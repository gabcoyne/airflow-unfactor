- **Analyze** Airflow DAGs into structured summaries
- **Convert** by refactoring DAG code into Prefect flow code with educational commentary
- **Validate** refactoring fidelity
- **Explain** Airflow concepts using Prefect equivalents
- **Organize** output into clean project structure for maintainable migrations

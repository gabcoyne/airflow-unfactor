# MCP Examples

## Claude / Cursor prompt

> Convert the DAG at `dags/my_etl.py` to a Prefect flow and generate tests.

## Output

- `flow_code` — Prefect flow
- `test_code` — pytest validation
- `mapping` — task mapping table
- `warnings` — any conversion caveats
- `dataset_conversion` — events, trigger snippets, and asset materialization scaffold
- `conversion_runbook_md` — post-conversion setup checklist for Prefect server/API configuration
